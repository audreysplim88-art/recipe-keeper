*Goal: Turn the recipe box into a real-time cooking companion*

| Feature | Status |
|---|---|

| Streaming conversational Claude API (`/api/sous-chef`) | 🔲 Planned |

| Conversational voice input (auto-send on silence) | 🔲 Planned |
